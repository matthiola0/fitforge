00為何選這個視覺

「F」的兩根橫桿就是啞鈴 — 圓形端點 = 槓片。
放大看是啞鈴、縮到 16px 仍是清楚的 F。

字面意義（健身工具）＋ 字母（品牌）合而為一，不靠任何文字輔助。圓形端點同時呼應 forge 火星、爐火的暖意 — 對應品牌色橙紅。

06SVG 源碼

單一 mark · 64×64 grid · fill="currentColor" 可繼承 CSS 顏色。

Mark (純圖標)

<!-- FitForge mark · 64×64 grid · stem + 2 arms + 2 plates -->
<svg xmlns="http://www.w3.org/2000/svg"
     viewBox="0 0 64 64"
     width="32" height="32"
     fill="#E14B36"
     role="img" aria-label="FitForge">
  <!-- stem -->
  <rect x="16" y="11" width="14" height="46" rx="7"/>
  <!-- top arm + plate (the upper "dumbbell") -->
  <rect x="16" y="11" width="24" height="14" rx="7"/>
  <circle cx="45" cy="18" r="8.5"/>
  <!-- mid arm + plate (the lower "dumbbell") -->
  <rect x="16" y="28" width="20" height="12" rx="6"/>
  <circle cx="40" cy="34" r="7"/>
</svg>

Wordmark (橫式)

<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 168 32" height="32">
  <!-- mark, scaled 0.5 to fit 32 -->
  <g fill="#E14B36" transform="scale(0.5)">
    <rect x="16" y="11" width="14" height="46" rx="7"/>
    <rect x="16" y="11" width="24" height="14" rx="7"/>
    <circle cx="45" cy="18" r="8.5"/>

        font-family="Inter, sans-serif"

</svg>
