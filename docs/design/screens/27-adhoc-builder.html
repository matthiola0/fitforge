DESIGN NOTES
  • Step 指示器：右上 mono「STEP 1 / 2」eyebrow + 下方 2-segment 進度 bar — 比圓點 dots 更節制、跟設計系統 mono 數字一致。Step 2 兩格都填滿、Step 1 只填第一格。
  • BodyPart chip（large multi-select）：56h × 圓角 28（pill）+ 22px padding + 18px font + 左側 20px check 圈。未選 = secondary 底 / border / 灰白 check 圈；selected = primary 底 + scale(1.05) + chip-up shadow + 反白 check ✓。複選明確、視覺重量足。沒用 icon — 7 個部位的字夠短可讀、icon 反而會顯得卡通。
  • Step 1 footer：顯示「2 / 7 部位」counter 給用戶確認；「全部清除」secondary button。CTA disabled 用 muted 底 + muted-fg/60 文字（不是純灰），保持暖色調 — 沿用設計系統「不純黑純白」。
  • Step 1 教練 hint（上下文敏感）：未選時用「上次練的是 X，今天換個部位吧？」幫用戶決策；已選時換成「肩 + 手臂 = 上推日的好組合，預估 45-55 分鐘」accent 底 + sparkles icon 給信心。沒選的時候不顯示時長預估 — 沒資料就別假裝有。
  • 已選部位 reminder（Step 2）：頂部小 chip row + 「改」link、accent peach 底 + 小 check icon。功能上等於 breadcrumb。
  • 動作數 stepper：6 顆 64×64 圓角方塊 mono 數字、水平排列（375 內勉強塞下 6 × 64 + 5 × 10 gap = 434、所以加上 overflow-x scroll 安全網）。預設 5 selected + 上方 corner 「推薦」mini pill — 給新手強烈預設。
  • 預估時長：stepper 下方 timer icon + 「45-55 分鐘 · ~15 組」— 時長 = 動作數 × 平均 9-11 分鐘的隱性公式，幫用戶感受 commitment。
  • 兩個 CTA 視覺權重對比：智慧推薦 = accent peach radial 漸層底（這頁是 PR 之外少數允許微漸層的位置，因為是 hero 分岔點）+ primary cta-arrow + sparkle 動畫 + 「建議」eyebrow 標籤；自己挑 = card 底 + outline + secondary cta-arrow。視覺上「智慧推薦」明顯更重，但「自己挑」沒有被埋。
  • 底部彩蛋：「重複上週這天的訓練」micro link — 給回頭客一個 zero-thought 入口、不搶 Step 2 兩個主要選擇。如果不要可拿掉。
  • 不做的事：沒用「上一頁 / 下一頁」雙按鈕模式 — back chevron 已在 header 處理。沒做 wizard 模式的全螢幕 transition placeholder（純靜態 mockup）。
