  • Delta chip 色系（遵守設計系統「不用紅綠燈」原則）：所有 delta 用 mono 等寬數字 + 灰底 chip + 方向 icon (arrow-up-right / arrow-down-right / minus)：
    • up（進步）：foreground 文字 — 強但不染色
    • down（退步）：muted-foreground 文字 — 弱化但不打紅
    • flat（持平）：muted 文字 + minus icon
    • pr（破紀錄）：唯一染 primary 的例外 — primary/.12 底 + primary 邊框 + primary 文字 + trending-up icon
  • 第一次做的動作：沒有上次可比、不顯示空 disclosure，改顯示「第一次做這個動作 · 建立基準 ✓」單行 — 給新手鼓勵、不是「找不到資料」。
