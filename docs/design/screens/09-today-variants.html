FITFORGE · HOME · STATE VARIANTS

首頁兩個變體.

Hero card 是首頁的「狀態指示器」 — 不換結構、只換內容。同樣 padding、同樣 radius、同樣位置。 A：訓練進行中 (resume)。B：還沒選課表 (empty state)。其餘區塊不變。

VARIANT A · 訓練進行中 · RESUME STATE

用戶離開 app 時訓練還沒結束 — 回來時 hero 顯示恢復點。進度條 + live pulsing dot， 次要連結「放棄訓練」用 muted 色 — 不嚇人、但需要兩次點擊 (有 confirm dialog)。

VARIANT B · 還沒選課表 · EMPTY STATE

Onboarding 跳過 / 用戶取消當前 plan 時。線條藝術插圖 取代 plan 資訊： bar + plate + spark (3 顆漂浮 dot)，呼應品牌 mark 但是 line-art 表現。次要連結 「先逛逛動作庫」給好奇心 / 還沒準備好決定的人一條出路。

「放棄」/「跳過」· 用 LINK 不用 BUTTON

負向動作降到 text link 級別、muted 色、小字、置中於主 CTA 下方。不用 destructive 紅 — 沒練完不是錯誤、不需要紅燈。

進度條 · SHIMMER 表達「進行中」

填滿部分有 highlight 從左掃到右 (2.4s 一輪)。低成本、不誇張、但讓「現在還在進行」這個事實在視覺層具現化。

SPACE 不空 · 替換不擴張

三個變體的 hero card 都用相同 height range — 不會因為「空狀態」就把整屏放空。維持版面節奏一致、下面的「本週節奏 / 最近訓練」也都顯示。
