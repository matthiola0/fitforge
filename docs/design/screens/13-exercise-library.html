FITFORGE · TAB 03 · 動作 (EXERCISE LIBRARY)

動作圖庫.

30 個 V1 MVP 動作。Sticky 搜尋區 + 水平 muscle filter + 2 欄方形卡片。第 1 張卡片 thumb 在 hover 時播放 scan-line 暗示 Lottie 動畫。 第三個 phone 示範 empty state。

STATE A · 預設 (30 個全部顯示)

「全部」chip 為選中、無搜尋字串。card grid 兩欄、可滾動。試著點不同 chip 觀察篩選即時生效。

STATE B · 空狀態 (搜尋無結果)

搜尋框輸入「跳繩」+ 「胸」filter — V1 沒有對應動作。Empty state 用 line-art search 插圖 + 教練口吻文案 + 「清除全部篩選」link，呼應 design system §08 voice rule「承認現狀 + 預告未來」。

STICKY 搜尋 · TITLE COLLAPSE

滾動到一定距離後 hero 大標題 hide、sticky 區塊 (搜尋 + filter chips) 維持頂部。第二支 phone 預先 scroll 到 collapsed 狀態示範該效果。

LOTTIE 縮圖 · MOTION 暗示

真實實作會用 Lottie autoplay-on-hover、靜止顯示第一幀。預覽用 hover 時的 scan-line + LOOP badge pulse 來暗示「這是動畫」、不是死圖。

2 COLUMN · NOT 3

375 寬度下選 2 欄而非 3 欄 — 每張卡 ≈ 167px、足夠塞下動作預覽 + 中英文 + chips。3 欄會讓中文名 truncate、體驗變差。
