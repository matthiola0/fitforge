Bar 解讀

64×8px track，fill 寬度 = 本次噸位 ÷ 本月最高噸位。 高於月均的 bar 用 primary 橙，低於月均用 foreground/30 灰； 垂直 dashed tick 是月均位置（不會跟著捲）。 下方 delta 用文字色階呼應，不另加箭頭 — 不要紅綠燈。

DESIGN NOTES
  • Header：「歷史」H2 · 28px，旁邊兩個 40px 圓形 icon button（搜尋 / 月曆 view 切換）。空狀態時兩 icon 變 muted/50 + cursor-not-allowed，因為還沒東西可搜。
  • Stats card：不平均分 3 等分 — 「本月訓練 8 次」用 42px num hero 強調主指標，旁邊放 success delta pill「+2 比上月」（success 色保留給「對比上月變好」這個語意）。下方總噸位 / 總時長用 18px secondary。
  • Sticky 月份：淺 hairline + 微 shadow 標示「黏在頂端」狀態；標題後綴月均噸位（小、normal tracking），右側顯示「5 / 8 次」當月達成度。
  • Row：3 欄 — 日期（48 寬，數字 24px 800、星期幾 caption） · plan 名 + meta（單行 mono num）· mini chart（64×8 + delta caption）。整 row 是 `<a>`，hover 換 secondary/50 底。
  • Mini chart：高 / 低於月均用 color 區分（primary vs foreground/30），dashed tick 標月均；delta 直接用「+34%」「−21%」字色呼應，不用箭頭、不用紅綠燈。
  • PR 標記：plan 名旁加 14px flame icon + primary 色 — 「這天有 PR」的最輕量標識，不喧囂。
  • Swipe-to-delete：row 左滑 86px，destructive 底色 + 「刪除」+ trash icon。Light 版第一筆刻意 pin 在 swiped 狀態示範。底下加 dashed 提示「左滑：刪除」+ mono 箭頭，幫 reviewer 一眼看懂。
  • Empty state：96 圓形 accent peach 容器 + dashed primary/30 邊 + 中央 calendar-days 40px icon，三顆 sparkle 環繞（2.8s loop）。文案沿用 §08「承認現狀 + 預告未來」公式：「完成第一次訓練後、這裡會開始長出你的進度」。CTA「開始第一次訓練 →」primary 52h，下方 ghost「看看新手指南」當輔線。
  • BottomNav：4 tab、歷史 active。Active 整個換 primary 色（icon + label + label bold），不另加底色 / 不另加 indicator dot — 色彩本身就是夠強的訊號。
