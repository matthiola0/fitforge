FITFORGE · CUSTOM PLAN · EDITOR

編輯課表.

V1 最複雜的表單。Sticky header + sticky save bar 之間是可滾動表單區。Inline editable day name、可拖拉重排的 exercise row、緊湊 stepper、bottom sheet 加動作。

STATE A · 編輯主畫面 (DIRTY)

有未儲存變更時、儲存按鈕亮起 primary glow、header 出現「已修改」warning dot 提示。取消按下後若 dirty 會彈 confirm modal (本頁不展開 modal、只示意 sticky bar)。

STATE B · 加動作 BOTTOM SHEET 展開

點 「+ 新增動作」開啟。Sheet 從底部滑上、約 85% 螢幕高、頂部 drag handle 可拖回。搜尋輸入 + 肌群篩選 chip + 動作 list。點動作 row 把它加入當前 day 並關閉 sheet。

STEPPER · COMPACT 22PX

22 × 22 -/+ 按鈕、中間 mono tabular 數字。三組 stepper (sets / reps / rest) 在一行內顯示。不用文字輸入、降低錯字機會、新手友善。

DIRTY 狀態指示

標題後 warning 色脈動 dot + 儲存按鈕 primary glow。返回時觸發 confirm modal (本頁示意、不展開)。dirty 用 warning 色 而非 destructive — 「需要注意」、不是「錯誤」。

BOTTOM SHEET · iOS PATTERN

不彈 modal、改用 bottom sheet — 拇指可及、context 不被覆蓋。Drag handle pill + 搜尋 + 篩選 + list 是 iOS / shadcn-vaul 標準。
