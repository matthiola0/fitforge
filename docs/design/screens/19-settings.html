DESIGN NOTES
  • Header：40×40 圓形 back button + 「設定」H4 字級（20px、left-aligned）+ 右上 avatar 28×28（accent peach）— sub-page chrome，沒有 BottomNav。
  • Profile teaser：頂部多放了 profile mini card（頭像 + 名字 + 訓練次數 + 連續天數 + 編輯入口）— 設定頁通常是用戶從頭像進來，第一行就看到自己的狀態，比直接跳設定 list 親切。如不要可刪。
  • Section card：每個 section 包一張 rounded-lg card + 1px hairline border。Row 之間用 1px border-top 分割（不是 padding gap）— shadcn 風 grouped list 是 row 緊貼。
  • Row icon：30×30 圓角方塊（border-radius sm + 1）+ secondary 底 + foreground 色 icon。不給每個 row 染不同顏色 — 那會變 Material 風。Destructive row 例外用 destructive/12 底 + destructive icon。
  • 控件選擇（最直接的）：
    • 主題 → chevron sheet（3 個選項：淺色 / 深色 / 跟隨系統）
    • 重量單位 → segmented [kg|lbs]（binary、語意對等、不適合 toggle）
    • 預設組間休息 → 行內 stepper（−/+ 26px 圓 + mono num + ±15s 增量）— 數值微調比開 sheet 快
    • 音效 / 震動 → iOS pill toggle 44×26（off/on 是真的 binary）
    • 語言 → locked，整 row 55% opacity + lock icon 取代 chevron + 「v2」 mini pill
  • Sub-label：row label 下方加 11.5px muted sub 解釋功能。「JSON · 含所有訓練、PR、設定」、「資料只存在你的裝置上」— 給新手安全感。語言 / 主題 / 單位這類自明的 row 不加。
  • 版本 row：「v1.0.0」mono + 「最新」success mini pill — 隱性檢查更新訊號。如果有新版本，這裡會換成「更新」primary pill + tap 進更新流程。
  • 清除所有資料：destructive 色限制只給這一處。sub label 同步染 destructive/70 「無法復原 · 會二次確認」— 預告會有 confirm modal，不會手滑炸資料。
  • Footer：「Made by [name] · 2026」+ 副標「Iron sharpens iron」（uppercase eyebrow 風）— 留 [name] 當 slot 待填。副標可拿掉。
