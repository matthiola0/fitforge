DESIGN NOTES
  • Hero：「訓練完成」用 primary 色、check-circle-2 icon 取代用戶 prompt 的 💪 emoji — 沿用 §08 PR 範例「兩個字 + Lucide 圖示」的成熟慣例。
  • Stat grid：3-col 等分、用 divide-x 區隔；mono + tabular-nums 確保數字不抖。
  • PR 卡：整份系統裡唯一同時用 glow + 微漸層的元件 — peach radial wash + primary ring + primary glow。Sparkle 4-6 顆四角星、錯開 0~1.8s 動畫，2.8s loop。
  • PR set row：整列換 accent / primary/.16 底色 + primary 文字、PR badge — 沿用 §02 set tracker 慣例。
  • 無 PR variant：把 PR 卡位置換成「本週累計」recap，給沒拿 PR 的訓練日一個正向 anchor — 連續訓練、累計噸位、本週進度。
  • Coach 一句話：底部 dashed border 卡片、像教練留紙條。PR 版「持續就是進步 + 下次 +2.5kg 建議」；無 PR 版「穩定 + 跳過組的修正建議」。不誇張、不罵人。
  • CTA：outline「產生分享圖」(48h) + primary「回首頁」(52h CTA size)、stacked。Light 版 sticky 在 viewport 底（向下捲時 CTA 仍可見）；mockup 為求展示完整內容暫關 sticky。
  • 沒做的事：沒放 BottomNav（takeover 結算頁，不在 tab flow）；沒放純綠色 success 圖標（primary 已經承擔「完成」語意，色彩焦點只有一個）。
