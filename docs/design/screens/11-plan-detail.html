FITFORGE · PLAN DETAIL · PRESET vs CUSTOM

課表詳情.

左：預設課表（不可編、可複製、2 天輪替）；右：自訂課表（可編、1 天）。兩者共用相同骨架，只差「預設徽章 / 主 CTA 文字 / 第二個 Day」三件事。Day accordion 預設展開、可點 header 收起。

VARIANT A · 預設 · ACTIVE

⭐ 預設徽章亮起、CTA 為 「複製為自訂」 + 「結束使用」 outline。⋯ menu 中有「複製」「匯出」 但沒有「刪除」(預設不可刪)。

VARIANT B · 自訂 · ACTIVE

沒有預設徽章、CTA 為 「編輯」 + 「結束使用」 outline。⋯ menu 中有「複製」「匯出」「刪除」(destructive、紅色)。

FOOTER 三態

Active + preset → 結束使用 + 複製為自訂。Active + custom → 結束使用 + 編輯。Inactive → 設為使用中 滿寬。本頁示範前兩種；inactive 模式延續 Plans list 進入點。

STICKY HEADER · BLUR

頂部 header 用 backdrop blur 浮在內容上、滾動時 title 仍可讀。標題在 hero 大字標題下方再次出現 — 是備援、不是冗餘 (滑下去之後 hero 不見、header title 接手)。
