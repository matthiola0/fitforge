動作動畫風格規範 Exercise Animation Style Guide

給未來動畫師、外包、AI 動畫工具的參考。教學用、看得出關鍵點、保持插畫感。 不要擬真錄影、不要把人畫死。30 個動作、同一個 character、同一套規則。

範圍 · 30 動作 · Lottie JSON 每支 · < 80KB · 3–4 秒 loop 輸出 · 60fps · 480 × 480 viewBox 本頁 SVG 為動畫意圖示意、非最終 deliverable

§2 · 環境 / 場景

純色 + 器材線稿 · 無健身房內裝
器材線稿庫

器材一律 純色塊或線稿、stroke-width: 3、所有端點 round。 重量片不畫斤兩數字、不畫品牌 logo。

§4 · 視角決策

看不到關鍵點的視角 = 沒用的動畫
視角 用於 看得到 例
45° 側面 ¾ 大部分動作 · 預設 軀幹角度 + 髖膝彎曲 + 一點深度 Squat · Bench Press · Deadlift · Row
90° 純側面 脊柱對齊是重點時 背是否平、髖角度 Plank · Hip Hinge · RDL
ISO 等角 需區分左右腳 / 不對稱 兩側肢體互不遮擋 Lunge · Bulgarian Split · Single-leg RDL
FRONT 前視 膝外展 / 內扣是重點 膝蓋外展軌跡 Sumo Squat · Goblet
BACK 背視 背部肌群 · 罕用 肩胛運動 Pull-up · Lat Pulldown

§5 · 標註系統

少而有用、不蓋住動作本身
關鍵點 · Dot

primary 實心圓、半徑 4px、標關節 / 觸地點

輔助線 · Dash

虛線 0.8px · primary 60% · 對齊參考

方向 · Arrow

在離心 phase 顯示、向心 phase 隱藏

啟動 · Pulse

標核心 / 主動肌、脈動 2s 一次

不加文字。 所有提示一律走外層 React overlay、由 i18n 翻譯。 動畫本體裡的 CORE / DOWN / UP 是本份 spec 的標註、實際 Lottie 不嵌字。

§6 · 時長 / 速度 / 檔案大小

每支 < 80KB · 3–4 秒 loop · 起終一致
單一 REP
3.0 – 4.0s · 比真實 tempo 慢約 1.3×、給新手看清楚
LOOP
第 N 幀必須與第 0 幀 像素一致 · 動畫師需用 onion-skin 驗證
速度切換
0.5× / 1× / 1.5× 三檔 · 用 Lottie playSpeed、不重做動畫
EASING
離心慢於向心 · cubic-bezier(0.45, 0, 0.55, 1) · 不用 linear
FPS
動態動作 60fps · 等長 hold（如 Plank）30fps 即可
viewBox
480 × 480 · 動作主體不超過中央 380 × 380 安全區
檔案大小
每支 < 80 KB · 30 動作 ≤ 2.4 MB 總量 · 不超過離線快取預算
顏色精度
用 hex 16 進位、不存 rgba() · 共享 swatch 由動畫師統一 import
瘦身策略 · 達到 80KB
  • ·用 shape、不用 image
  • ·骨架共用、各動作只改 keyframe
  • ·路徑點 ≤ 6 / 段 · 不用毛邊曲線
  • ·輸出走 lottiefiles --minify
  • ·不嵌字體 · 不嵌 PNG raster
無障礙
  • ·尊重 prefers-reduced-motion · 自動停在 mid-rep 靜態 frame
  • ·提供「下一步示意圖」靜態 PNG fallback
  • ·aria-label 由 React overlay 提供、動畫本身 aria-hidden

§7 · 該 / 不該

DO
  • 讓動作起 / 終姿勢可辨識、即使靜止
  • 主動肌位置加 pulse、讓新手知道「哪裡該酸」
  • 離心 phase 慢、向心 phase 略快
  • 所有 30 個動作用同一個 character、同一個鏡頭距離
  • 器材線稿用同一個 stroke 粗細
DONT
  • 不畫汗珠、肌肉抖動、表情
  • 不畫品牌 logo / 重量數字 / 健身房
  • 不用真人 rotoscope / motion capture trace
  • 不畫錯誤示範與正確並列（另外做 EX-XXX-bad.json）
  • 不用 gradient · 不用 shadow · 不用模糊

§8 · LottieFiles 採購關鍵字

若自製成本過高、可採購後依本 spec 重繪

關鍵字組合策略：動作名 + 風格詞 + 視角詞 + 排除詞。 建議優先 flat illustration 與 minimal、避開 3d 與 cartoon mascot。

動作名 · ENGLISH
EXsquat EXbench press EXdeadlift EXplank EXpushup EXpullup EXlunge EXrow EXshoulder press EXbicep curl EXtricep extension EXlat pulldown EXRDL EXhip thrust EXleg press
風格詞 · STYLE
+flat illustration +minimal +character animation +fitness icon +workout illustration +line art +gender neutral +limited palette
視角詞 · ANGLE
+side view +3/4 view +isometric +profile
排除詞 · NOT
−3d −realistic −cartoon mascot −anime −chibi −muscle bro −gradient −logo
範例 query · 好
squat flat illustration side view minimal -3d -realistic -gradient
範例 query · 不好
workout man cool animation muscle

採購後 必須 依本 spec 重繪 character 與配色、不可直接使用原作者角色 — 確保 30 個動作視覺一致。
