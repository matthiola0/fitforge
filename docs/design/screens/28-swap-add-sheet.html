DESIGN NOTES
  • Sheet shell：底部滑入、佔 ~80%（680/844）。drag handle 36×4 / muted-fg/40 / 圓角 4px、置中。close X 在右上 32×32 圓形 secondary 底，避免跟 drag handle 搶軌道。
  • Backdrop：用 foreground 顏色 @ 45% opacity 當 scrim（不是 background/95 — 那會等於遮掉背景失去 sheet 漂浮感）。上方 status bar 保留實色不被 dim。
  • Item card：48×48 Lottie 縮圖佔位 — 對角線 muted 條紋底 + 中央 Lucide icon + 右上 14px play triangle 表示「這裡是動畫」。陪襯資訊用 chip row：muscle chip = secondary 中性、器材 chip = accent peach + 小 icon、雙肌群顯示「+1 後三角」省略 chip。
  • Tabs（依部位 / 依肌群 / 搜尋）：shadcn 風 segmented control。bg-muted track + active 換 card 底 + ds-sm shadow + 文字 700。每個 tab 帶 14px Lucide icon 區分。
  • 依部位：chip-md（32h、14px rounded）水平 wrap 排列 7 個部位，active 整顆換 primary。下方直接顯示動作（最多 3 個 + 「查看全部 12 個」入口），避免長 list overload。
  • 依肌群：bodyPart 用 collapsible card group（chevron-right / chevron-down 區分狀態）。展開後 muscle chip 用較小的 chip-muscle（28h），active 用 accent peach + primary border，比依部位的 active primary 弱一點 — 這層是「過濾條件」不是「導航焦點」。
  • 空狀態（Swap）：因為 Tier 1（同主肌群）通常很少 / 偶爾 0 個，這個狀態實際會發生。用 search-x icon、accent dashed 圓、文案沿用 §08「承認現狀 + 給可行下一步」公式 + 主 CTA「放寬到同部位」secondary 變體 + ghost「進動作庫」次選。
  • 設定組數二級 sheet：從父 Add Sheet 內部往上推、佔 ~66%（560/844）。父 sheet 在後面留 96px peek（標題模糊可見），給「往上堆疊」的視覺暗示；back chevron + 「回新增動作」label 在頂部，讓返回路徑明確。
  • 三 stepper 並排：375 內寬 / 3 = 約 117 per card。組數用 segmented [3][4][5]（離散選項）、次數用 8─12 range + 兩組 −/+（min / max 各一），休息用單值 + −/+（±15s 增量）。三個 metaphor 不同但對齊 grid。下方加預覽 pill 「3 sets × 8-12 reps · rest 90s」，讓所有設定一行讀完。
  • 新手教練語氣：二級 sheet 含 dashed border 教練提示「新手建議 3 × 8-12。重量挑到第 8 下開始吃力。」— 不只設定數字、也告訴為什麼。沿用 §08「直接、像私人教練」voice。
  • 底部 CTA：取消（outline、flex-1）+ 加入（primary、52h、flex 1.6）— 用 flex 權重而非寬度區分主次，主 CTA 視覺更重但兩者都全寬可點。
